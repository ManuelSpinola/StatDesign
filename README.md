# StatDesign

Herramienta interactiva para el diseño de estudios y muestreo en ciencias ambientales y recursos naturales.

Parte de la suite StatX junto con StatFlow y StatModels.

## Módulos
- Tipos de estudio
- Diseños de muestreo + calculadora de tamaño de muestra
- Asistente de diseño
