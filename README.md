# StatDesign <img src="inst/app/www/hexsticker_StatDesign.png" align="right" height="139"/>

**StatDesign** es una plataforma interactiva para diseño de estudios y muestreo, parte del ecosistema [StatSuite](https://github.com/ManuelSpinola). Diseñada para enseñanza e investigación en ecología y ciencias de la biodiversidad.

## Módulos disponibles

| Módulo | Descripción |
|--------|-------------|
| Tipos de estudio | Tipos de estudios científicos y su aplicación |
| Diseños de muestreo | Métodos y estrategias de muestreo |
| Fundamentos del diseño | Principios del diseño de estudios |
| Asistente | Guía interactiva para elegir el diseño correcto |
| Ayuda | Documentación y referencias |
| Acerca de | Información del proyecto |

## Instalación

```r
install.packages("remotes")
remotes::install_github("ManuelSpinola/StatDesign")
```

## Uso

```r
library(StatDesign)
StatDesign::run_app()
```
